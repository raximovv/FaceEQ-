# FaceEQ
What it does
Tracks your face via webcam using MediaPipe. Turn your head left or right to rewind or fast-forward a track. Tilt up or down to change playback speed. Your head becomes a rotary encoder.

**How to run**

Drop an audio file named track.wav or track.mp3 in the folder
python day05_faceeq.py

**Controls
**
Turn head LEFT = rewind
Turn head RIGHT = fast-forward
Tilt head UP = speed up
Tilt head DOWN = slow down
Press SPACE = play / pause
Press R = reset to start
Press Q = quit
